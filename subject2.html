Computers Quiz - Learn and Test Your Knowledge

Who invented the first mechanical computer?

  • Alan Turing
  • Charles Babbage
  • Konrad Zuse
  • Charles Babbage
The correct answer is Charles Babbage.
Charles Babbage is known as the father of the computer for inventing the first mechanical computer, the Analytical Engine.

What is the primary function of the CPU in a computer?

  • Store data
  • Manage power
  • Display graphics
  • Execute instructions
The correct answer is Execute instructions.
The CPU (Central Processing Unit) executes instructions from the computer's programs, essentially acting as the brain of the computer.

Which of the following is an example of an open-source operating system?

  • Windows
  • macOS
  • Android
  • Linux
The correct answer is Linux.
Linux is an open-source operating system, meaning its source code is available to the public to use, modify, and distribute.

Who is the founder of Microsoft?

  • Steve Jobs
  • Mark Zuckerberg
  • Larry Page
  • Bill Gates
The correct answer is Bill Gates.
Bill Gates co-founded Microsoft in 1975, which has since become one of the world's largest technology companies.

What does HTTP stand for?

  • Hyper Text Transfer Protocol
  • High Transmission Text Protocol
  • Hyperlink Transfer Protocol
  • HyperText Transfer Protocol
The correct answer is HyperText Transfer Protocol.
HTTP is the protocol used for transferring web pages over the internet.
